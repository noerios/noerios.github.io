The Barber Booker

Check it

A same-day booking platform for barbers.

Built for barbers who need a simple and effective online booking system. This application is scalabe to apply to personal services of all kinds.

The front end of this application utilizes a Bootstrap framework, styled with custom CSS. On the back end, we have a MySQL database storing user data and booking times. Passport.js provides a user authentication middleware for Node. Express-session stores session data. Find the code on GitHub.
